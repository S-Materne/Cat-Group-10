# System Requirements

## Functional Requirements

The system must:

- Allow service selection
- Allow client name entry
- Allow quantity entry
- Allow price entry
- Automatically calculate total
- Save data in MySQL
- Display all records

---

## Non-Functional Requirements

- Must run in browser
- Must use MVC structure
- Must connect to database successfully
- Must execute without errors

---

## Software Requirements

- XAMPP
- PHP 8+
- MySQL
- Web Browser
