## Project Description

The Musanze Digital Service Desk is a browser-based MVC web application designed to:

- Capture service requests
- Automatically calculate total cost
- Store records in MySQL database
- Display all submitted records

---

## Services Included

1. Potato Order
2. Event Ticket
3. Home-stay Booking

---

## Objective

To demonstrate how digital systems can replace manual record keeping using MVC architecture.
