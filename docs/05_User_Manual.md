# User Manual

## How To Install

1. Copy project folder to htdocs
2. Start Apache and MySQL
3. Import database/schema.sql
4. Visit:

http://localhost/CAT_GROUP_2_FEB_A/public/

---

## How To Use

1. Click "Create New Request"
2. Select service type
3. Enter client name
4. Enter quantity
5. Enter price
6. Total calculates automatically
7. Click Save
8. View record in table

---

## Notes

- All records are stored permanently.
- Total is automatically calculated.
